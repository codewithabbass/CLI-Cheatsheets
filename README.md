## 🚀 Quick Start

1. **Browse** a category from the table above.
2. **Find** the command you need.
3. **Copy** and paste it directly into your terminal.

Each command entry follows this format:

```
Command:     The exact command to run
Description: What it does
Example:     Real-world usage
Output:      Expected result (where applicable)
```
